## What to build

Update the **Homepage** calls to action so the public entry point routes visitors into the correct open experiences. The primary action should invite students to join an activity, while the secondary teacher action should let a teacher host an activity.

## Acceptance criteria

- [ ] Homepage primary CTA is "Join activity" and links to `/student`.
- [ ] Homepage secondary teacher CTA is "Host activity" and links to `/teacher`.
- [ ] CTA copy avoids sign-in, account, dashboard, or portal language.
- [ ] Existing Homepage SEO/public-entry behavior remains intact.
- [ ] Tests or a focused render check cover the CTA labels and links.
